# Eufy Add-ons

Private/personal Home Assistant add-on repository.

## Add-ons in this repository

- **eufy_lock_bridge** — HTTP bridge for controlling Eufy locks (garage +
  side door, T85D0 model) that aren't fully working through
  `eufy-security-ws` due to an account-wide DSK cloud-handshake issue. See
  [eufy_lock_bridge/README.md](eufy_lock_bridge/README.md) for setup and
  usage details.

## Installing this repository in Home Assistant

1. In Home Assistant: **Settings → Add-ons → Add-on Store**
2. Click the three-dot menu (top right) → **Repositories**
3. Paste this repo's URL and click **Add**
4. Close the dialog, refresh — **Eufy Lock Bridge** appears under a new
   section using this repository's name
5. Click it → **Install**
